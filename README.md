# Endfield Substrate Plan

武器查询工具，用于筛选和查找游戏中的武器数据。

## 数据结构

```
data/
├── weapons.json      # 武器数据（61把武器）
├── dungeons.json     # 副本数据（6个副本）
├── filters.json     # 筛选器选项
└── type-icons.json  # 武器类型图标映射
```

如需更新数据，直接修改对应JSON文件即可，无需触碰index.html。

## 功能

### 武器筛选
通过三个属性位置筛选武器：
- **位置一（S1）**：力量提升、敏捷提升、智识提升、意志提升、主能力提升
- **位置二（S2）**：攻击提升、暴击率提升、伤害提升等
- **位置三（S3）**：压制、迸发、夜幕等

筛选逻辑：位置内多选为OR，位置间为AND。实时显示命中的武器数量。

### 武器详情
点击武器卡片查看详情：
- 武器名称、稀有度、类型
- 三条属性
- 角色适配建议（如有）

### 获取地点分析
选择武器后，展示可获取该武器的地点（副本），以及使用最优S1组合框选时的可能命中武器。

**框选规则**：
- S1固定选择3个属性（所有副本通用）
- S2或S3只选1个属性（不同副本有不同属性池）
- 在同一副本内，框选后随机获得该副本内符合条件的武器

系统自动计算每种S1组合的命中结果，过滤掉子集结果，只展示独立的最优组合。

## 数据统计

- 61把武器（6星30把、5星21把、4星10把）
- 6个副本地点
- 5种武器类型：单手剑、双手剑、手铳、长柄武器、施术单元

## 部署

纯静态网页，可直接用浏览器打开 `index.html`，或部署到任意静态托管服务（Vercel、Netlify、GitHub Pages）。
